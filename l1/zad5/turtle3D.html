Choose number of sides of polygon

Controls:

  • W, S, A, D - move
  • Arrow Keys - rotate left, right, up, down
  • Q, E - roll left, right
  • Left click to enable or disable rotation with mouse

Every command should end with semicolon ';'

  • fw x; - move x forwards
  • bw x; - move x backwards
  • rt x; - rotate x degrees clockwise
  • lt x; - rotate x degrees counterclockwise
  • dt x; - rotate x degrees downwards
  • ut x; - rotate x degrees upwards
  • repeat x [ commands ] - repeat commands x times no need to put ';' after ']'
  • up; - pen up, stop drawing
  • down; - pen down, start drawing
